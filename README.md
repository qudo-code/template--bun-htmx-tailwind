# bun + htmx + tailwind

## Getting Started 
Install [Bun](https://bun.sh/docs/installation) if you don't already have it.

### Install Project
```bash
bun install
```

### Run App
```
bun start
```

### Dev App
```
bun run dev
```

This project was created using `bun init` in bun v0.6.3. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
